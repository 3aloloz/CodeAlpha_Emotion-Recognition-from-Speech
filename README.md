# 🎙️ Speech Emotion Recognition using Deep Learning

A Deep Learning project for recognizing human emotions from speech audio using the RAVDESS dataset.

## Project Overview
This system classifies speech into emotions such as:

- Neutral
- Calm
- Happy
- Sad
- Angry
- Fearful
- Disgust
- Surprised

## Dataset
RAVDESS (Ryerson Audio-Visual Database of Emotional Speech and Song)

## Features
- Log-Mel Spectrogram
- Audio Augmentation
- Deep CNN
- Confusion Matrix
- Accuracy / Loss Visualization
- Real-time prediction support

## Tech Stack
- Python
- TensorFlow / Keras
- Librosa
- NumPy / Pandas
- Matplotlib / Seaborn
- Streamlit
